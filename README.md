## Setup

- Enter the dimensions of your primary monitor

  ```
   set SCREEN_WIDTH to <width>
   set SCREEN_HEIGHT to <height>
  ```

- Run the following

  ```console
  $ osascript DesktopBrickBreaker.applescript
  ```
